Загрузка каталога из Excel

Поддерживается только `.xlsx`. Импорт обновляет найденные товары, создает новые категории по колонке `Категория`, а позиции, которых в файле больше нет, делает неактивными.

.xlsx only

Обязательные колонки

Название Базовое имя товара. Если заполнена `Толщина`, система сама добавит к имени `4 мм`.
Цена Число без валютного текста. Пример: `14500` или `14500.50`.
Категория Название раздела витрины. Именно сюда товар попадет после импорта.

Необязательные колонки

Толщина Число в мм. Пример: `4`. На витрине получится имя вроде `Сатин светлый 4 мм`.
Формат Максимальный размер строго в формате `ширина*длина`, например `3210*2250`.
Фото / Image / Photo Можно дать URL или hyperlink из Excel. Если колонки нет, товар просто останется без фото.
